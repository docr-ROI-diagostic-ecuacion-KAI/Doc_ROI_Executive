# DocROI Executive

Landing estática de DocROI Executive para captar solicitudes de:

- formadores y trainers para universidades, escuelas de negocio y programas executive;
- diagnóstico e información para formación in-company.

## Despliegue en Vercel

Este repositorio no necesita build. Vercel puede publicarlo como proyecto estático usando `index.html` como entrada principal.

## Archivos principales

- `index.html`: versión principal para producción.
- `docroi-executive.html`: copia navegable directa de la landing.
